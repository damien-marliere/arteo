Connexion par mot de passe d'application

Gmail : comment obtenir un « mot de passe d'application »

  1. Activez la validation en deux étapes : myaccount.google.com/security.
  2. Allez sur myaccount.google.com/apppasswords.
  3. Donnez un nom (ex. « Artéo ») puis Créer.
  4. Copiez le code de 16 caractères.
  5. Collez votre adresse Gmail et ce code ci-dessus, puis Enregistrer et Tester.
